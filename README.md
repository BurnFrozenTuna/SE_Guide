# SE_Guide — Fish_In_the_Canvas

An unofficial starter guide to the StoryEngine Canvas, written by **Frozen_Tuna**.

**📖 Read it here:** https://github.com/BurnFrozenTuna/SE_Guide/edit/main/README.md

---

## What this is

A node-by-node walkthrough of the StoryEngine Canvas for creators who are new to it —
what each node is, what you actually write inside it, and how it behaves under the hood.

It covers the story, character, lorebook, image, variable, trigger and achievement nodes,
plus Advanced Settings and the publish page, with screenshots throughout.

## What this isn't

**Official documentation.** This was put together while actually building on the Canvas,
so some of it may be wrong, and parts of the platform's internals are deliberately
undisclosed. Numbers — file size caps especially — get changed by the platform fairly
often. Keep the Canvas open and read this alongside it.

## Found a mistake?

Please tell me. Open an [issue](../../issues) with what's wrong and, if you have one,
a screenshot. Corrections are genuinely welcome — that's how most of what's in here
got worked out in the first place.
