## What this is

A node-by-node walkthrough of the StoryEngine Canvas for creators who are new to it —
what each node is, what you actually write inside it, and how it behaves under the hood.

It covers the story, character, lorebook, image, variable, trigger and achievement nodes,
plus Advanced Settings and the publish page, with screenshots throughout.

## Again, because people skim: this is not official

Everything here was worked out **by hand, while building**. Parts of the platform's
internals are deliberately undisclosed, so some of this is guesswork, and some of it
is simply wrong. Numbers — file size caps especially — get changed by the platform
without notice.

**Do not take this as a source of truth.** Keep the Canvas open, check what it tells
you, and believe the Canvas over this document every single time.

## Found a mistake?
